# EEG Signal Analysis & Seizure Detection

This project focuses on processing EEG signals using Python. It includes automated detection of epileptic spikes and advanced frequency filtering.

## Features
- **Signal Simulation:** Generates synthetic EEG data with embedded seizure activity.
- **Spike Detection:** Automated counting of abnormal voltage spikes using threshold logic.
- **Time-based Analysis:** Filtering results based on specific time windows (e.g., first second of recording).
- **Advanced Filtering:** Implementation of Butterworth Band-pass filters to extract specific brain waves (Alpha, Beta).

## Libraries Used
- `NumPy`: For numerical computations and array handling.
- `SciPy`: For signal processing and filtering.
- `Matplotlib`: For data visualization.

## How to use
Run the `signal_analysis.py` script to see the analysis results and signal plots.
